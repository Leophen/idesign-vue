# Divider 分割线

分割线是一个呈线状的轻量化组件，起到分割、组织、细化的作用，用于有逻辑的组织元素内容和页面结构。

## 基本用法

可通过 `dashed` 属性来控制分割线样式。

::: demo
divider/demo1
:::

## 带文字的分割线

可通过 `align` 属性来控制分割线文字内容的位置。

::: demo
divider/demo2
:::

## Divider API

| 属性   | 说明       | 类型                      | 默认值   |
| ------ | ---------- | ------------------------- | -------- |
| align  | 文本位置   | `left"〡"right"〡"center` | `center` |
| dashed | 是否为虚线 | `boolean`                 | `false`  |
